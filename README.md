# MULTILAYER TOOL

<!--[![DIO](/doc/images/DOI.svg)](https://www.nature.com/articles/s41598-018-28948-z)-->

## Description

A powerfull tool for visualising, navigate and analysing spatial transcriptomics.

## Run

Launch : `python3 Multilayer.py`
 
## Dependencies

###### - Numpy

```bash
pip install numpy
```


# MULTILAYER COMPRESSOR

## Description

For high resolution dataset, we developed module for compress data. Multilayer compressor is able to merge several gexels in one big gexel.

## Run

Launch : `python3 Multilayer_Compressor.py`

## Dependencies

###### - Numpy

```bash
pip install numpy
```

###### - Pandas

```bash
pip install pandas
```
